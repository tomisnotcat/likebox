# LikeBox - 产品点赞平台

一个让用户为自己喜欢的产品点赞的社区平台。

## 功能

- 用户注册/登录
- 产品浏览、搜索、分类
- 为产品点赞
- 评论产品

## 本地运行

```bash
npm install
node server.js
```
访问 http://localhost:3000

## 部署到 Vercel

1. 安装 Vercel CLI: `npm i -g vercel`
2. 进入项目目录: `cd likebox`
3. 部署: `vercel`

或者连接到 GitHub，直接从 GitHub 部署：

1. 把代码推送到 GitHub
2. 访问 vercel.com，导入项目
3. 一键部署

## 技术栈

- 前端: HTML + TailwindCSS + Vue 3
- 后端: Node.js + Express
- 存储: JSON 文件 (适合中小型项目)
